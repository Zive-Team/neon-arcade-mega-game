# Neon Arcade (neon-arcade-mega-game)

Neon Arcade（ネオン・アーケード）は、PCやスマートフォンのブラウザからインストール不要で、いつでも手軽にカジュアルゲームを楽しめるレトロネオン調のゲームプラットフォームです。

海外メガテックに依存しないデジタル経済圏を目指す「zive（ゼィヴ）」構想の一環として、自由なオープンソース（OSS）の精神に基づいて開発されています。

## ✨ 主な特徴

- **レトロネオン調のUI**：80年代のゲームセンターをイメージした、サイバーで鮮やかなデザイン。
- **マルチデバイス対応**：PCだけでなく、スマートフォンのタッチ操作にも最適化されています。
- **完全無料・登録不要**：面倒なインストールや会員登録なしで、1タップで起動します。
- **オープンソース**：コードをすべて公開しており、誰でも仕組みを学んだりカスタマイズしたりできます。
